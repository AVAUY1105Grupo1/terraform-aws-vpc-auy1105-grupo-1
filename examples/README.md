# Ejemplos de uso del Módulo de Redes

## Objetivo del ejemplo
Esta carpeta tiene como propósito guiar de manera práctica en la implementación del módulo de redes en un entorno aislado. Demuestra cómo declarar correctamente los parámetros requeridos para desplegar una topología de red funcional en AWS.

## Instrucciones de ejecución
1. Accede al directorio del ejemplo:
```
   cd examples/despliegue_completo
```
2. Ejecuta los comandos estándar de inicialización y despliegue:
```
  terraform init
```
```
  terraform plan
```
```
  terraform apply
```
